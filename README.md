#  Crypto Wallet

Self-custody crypto wallet. One file. No server. No KYC.

## Features
- 10 blockchains: BTC, ETH, SOL, BNB, XRP, DOGE, AVAX, LTC, TRX, ICP
- AES-256-GCM encrypted seed phrase (same standard as MetaMask)
- Private keys derived on-device, never stored
- Works from USB drive on any computer with internet
- Dark/light mode, PWA-ready, mobile-friendly

## Usage
Download `crypto-wallet.html` and open in any modern browser.

## Security
- Seed phrase encrypted with AES-256-GCM
- Private keys never leave your device
- No server, no tracking, no accounts

## Warning
This is experimental software. Do not store large amounts.
Always back up your seed phrase offline.
